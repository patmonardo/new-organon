# Meta-Cognition: Pure Reason Speaking for Itself

## The Key Insight

**Cognition having an external beginning tells us that Pure Reason speaking for itself is not cognition, but meta-cognition.**

## The Distinction

**Cognition:**
- Has an external beginning
- Not self-determining (in this sense)
- Starts from outside
- Finite cognition

**Meta-Cognition:**
- Pure Reason speaking for itself
- Self-determining
- No external beginning
- The Idea recognizing itself

## Why This Matters

1. **Cognition** = Has external beginning = Not Pure Reason speaking for itself
2. **Pure Reason speaking for itself** = Meta-cognition = Not cognition
3. **Meta-cognition** = Cognition about cognition = Self-recognition
4. **The Idea** = Where meta-cognition happens
